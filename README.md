Iris Classifier Project by Francis Makombe



Overview



This exciting project uses machine learning to perform the classification of iris flower species based on sepal and petal measurements.



Research Question



Can machine learning be used to accurately classify iris flower species using flower measurements?



Dataset

Built-in Scikit-learn Iris dataset

150 flower samples

Species:

Setosa

Versicolor

Virginica

Python

Pandas

Methodology

Loaded Iris dataset from Scikit-learn

Converted data into a Pandas DataFrame

Split data into training and testing sets

Applied KNN classification model

The KNN machine learning model successfully classified iris flower species with perfect predictive accuracy, demonstrating that sepal and petal measurements are highly effective predictors.
